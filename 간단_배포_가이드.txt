방법 1: 배치 파일 실행 (가장 쉬움!)
================================================================================

1. "push_to_github.bat" 파일을 더블클릭

2. 엔터 누르고 기다리기

3. GitHub 로그인 창이 뜨면 로그인

4. ✅ 업로드 완료!

5. https://streamlit.io/cloud 접속
   → Continue with GitHub
   → New app
   → Repository: leeh1149/OUTLETDASHBOARD
   → Main file: dashboard_streamlit.py
   → Deploy!

6. 🎉 완료!

================================================================================
방법 2: GitHub Desktop 사용
================================================================================

1. GitHub Desktop 다운로드: https://desktop.github.com

2. File → Clone repository
   → URL: https://github.com/leeh1149/OUTLETDASHBOARD.git
   → Clone

3. 다음 파일들을 복사:
   원본: C:\Users\AD0581\Documents\ai study\
   대상: C:\Users\AD0581\Documents\OUTLETDASHBOARD\
   
   파일:
   - dashboard_streamlit.py
   - requirements.txt
   - README.md
   - .gitignore

4. GitHub Desktop에서:
   → Commit to main
   → Push origin

5. https://streamlit.io/cloud 접속
   → Continue with GitHub
   → New app
   → Repository: leeh1149/OUTLETDASHBOARD
   → Deploy!

6. 🎉 완료!

================================================================================
문제 해결
================================================================================

Q: "git is not recognized" 오류
A: Git 설치 필요 → https://git-scm.com/download/win
   또는 GitHub Desktop 사용

Q: 업로드는 됐는데 Streamlit Cloud에서 앱이 안 보여요
A: https://streamlit.io/cloud 에서 로그아웃 후 다시 로그인

Q: 배포 실패
A: Streamlit Cloud에서 "Logs" 확인
   - 대부분 requirements.txt 문제
   - 파일 경로 문제
