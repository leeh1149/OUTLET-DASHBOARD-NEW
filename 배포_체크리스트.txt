📋 배포 단계:

1️⃣ GitHub 저장소 생성
   - [ ] GitHub.com 접속 및 로그인
   - [ ] "New repository" 클릭
   - [ ] 저장소명: dx-outlet-dashboard
   - [ ] Public으로 설정
   - [ ] "Create repository" 클릭

2️⃣ 파일 업로드 (3가지 방법 중 선택)
   
   방법 A: GitHub 웹사이트 직접 업로드
   - [ ] 저장소 페이지에서 "uploading an existing file" 클릭
   - [ ] 다음 파일들 업로드:
     * dashboard_streamlit.py
     * requirements.txt
     * README.md
     * DX OUTLET MS DB.csv
     * STREAMLIT_CLOUD_배포_가이드.md
   - [ ] "Commit changes" 클릭

   방법 B: Git 설치 후 자동화 스크립트
   - [ ] Git for Windows 설치: https://git-scm.com/download/win
   - [ ] 배포_자동화_스크립트.bat 실행
   - [ ] GitHub 저장소 생성 후 제공된 명령어 실행

   방법 C: GitHub Desktop 사용
   - [ ] GitHub Desktop 설치: https://desktop.github.com/
   - [ ] 새 저장소 생성 (Public)
   - [ ] 파일들 추가 후 "Publish repository"

3️⃣ Streamlit Cloud 배포
   - [ ] https://share.streamlit.io/ 접속
   - [ ] GitHub 계정으로 로그인
   - [ ] "New app" 클릭
   - [ ] 배포 설정:
     * Repository: dx-outlet-dashboard
     * Branch: main
     * Main file path: dashboard_streamlit.py
   - [ ] "Deploy" 클릭

4️⃣ 배포 확인
   - [ ] 배포 완료 후 제공된 URL 접속
   - [ ] 대시보드 정상 작동 확인
   - [ ] 모든 기능 테스트

- CSV 파일이 100MB를 초과하면 Git LFS 사용 필요
- API 키는 Streamlit Cloud Secrets에 등록
- Public 저장소 필수 (무료 배포를 위해)
